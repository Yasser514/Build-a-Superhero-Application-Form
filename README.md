Superhero Application Form

A React-based web application that allows users to apply as a superhero by submitting personal details, power origins, and abilities through an interactive form.

🚀 Features
Input fields for:
Hero Name
Real Name
Dropdown selection for power source (e.g., mutation, technology, magic)
Multi-select checkbox system for superhero abilities
Form validation (disables submit until all required fields are filled)
Dynamic state management using React hooks
🛠️ Technologies Used
React (useState hook)
JavaScript (ES6+)
HTML5
CSS3
📂 Project Structure
Build-a-Superhero-Application-Form/
│── index.html
│── styles.css
│── index.jsx
⚙️ How It Works
Uses React useState to manage:
Hero name
Real name
Power source
Selected powers
Handles checkbox selections dynamically using arrays
Disables the submit button until all required fields are completed
Updates UI based on user input
📦 Installation & Setup
Option 1: Run in Browser
Clone the repository:
git clone https://github.com/Yasser514/Build-a-Superhero-Application-Form.git
Open index.html in your browser
Option 2: Run with React (Recommended)
Create a React app:
npx create-react-app superhero-form
cd superhero-form
npm start
Move logic into src/App.js
